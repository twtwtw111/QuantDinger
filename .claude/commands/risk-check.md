# /risk-check — 组合风险评估

## 用途
评估当前所有运行中策略的整体风险敞口，给出仓位建议和风险警告。

## 使用方式
```
/risk-check
```
无需参数，自动拉取当前运行中的策略和持仓数据。

## 执行步骤

1. 查询所有运行中策略：`GET /api/agent/v1/strategies`（filter: status=running）
2. 查询当前持仓：`GET /api/quick-trade/position`
3. 查询账户余额：`GET /api/quick-trade/balance`
4. 计算各策略仓位占比
5. 评估策略间相关性（同品种/同方向 → 高相关）
6. 调用 risk-manager agent 输出完整风险报告

## 触发警告的阈值

| 指标 | 警告阈值 | 危险阈值 |
|------|---------|---------|
| 单策略仓位占比 | > 8% | > 12% |
| 同品种总敞口 | > 15% | > 25% |
| 总持仓比例 | > 60% | > 80% |
| 单日浮亏 | > 1.5% | > 3% |
